# Proyecto Práctica Profesionalizante

Proyecto 1er. año Ciencia de Datos e Inteligencia Artificial

## Intro

Cada carpeta contiene un script python que permite leer archivos csv de un directorio determinado
y con un formato predeterminado con información del uso de la tarjeta SUBE.

La carpeta visor contiene una pequeña clase para visualizar archivos pdf basada en pillow/PyMuPDF
y un script de ejemplo que la utiliza.

## Requerimientos: 
*   Python 3.14.0+
*   matplotlib 3.10.7+ 
*   numpy 2.3.4+ 
*   pandas 2.3.3+
*   pillow 12.0.0+ 
*   PyMuPDF 1.26.7+ 
*   scikit-learn 1.7.2+

## Instalación: 
*   Instalar python 
*   desde linea de comandos ejecutar: pip install pandas numpy scikit-learn matplotlib pymupdf pillow
*   descargar proyecto, descomprimir csv.7z
*   ejecutar python main.py o desde vscode
